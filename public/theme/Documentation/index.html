Upload Files
Info : To upload a file through cPanel's File Manager, please do the following:

Also you will see 6 directories, css, js, fonts, img, vendor, phpmailer directories.

  1. Extract the received zip file
  2. Open "main-files" folder.
  3. Login to cPanel and open the File Manager
  4. Navigate to the folder where you want the upload to be located
  5. Click on the Upload icon from the top toolbar
  6. Click the Browse box (Some web browsers will have a Choose File box instead Locate the file on your computer that you wish to upload
  7. Once the file has been selected, it will automatically upload Now hit the path where your folder is located
